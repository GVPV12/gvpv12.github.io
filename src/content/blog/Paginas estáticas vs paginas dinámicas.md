[[Astro]]

Una página web estática es una ==página que no cambia su contenido dinámicamente según la interacción del usuario==. Su contenido es fijo y se entrega al navegador como un archivo HTML, CSS y JavaScript pre-creado. En contraste con las páginas web dinámicas, que generan el contenido en tiempo real, las páginas estáticas son más sencillas de crear y mantener, y suelen ser más rápidas de cargar. 

Características clave:
- **Contenido fijo:**
    El contenido de una página estática es el mismo para todos los usuarios y no cambia a menos que se modifique manualmente. 
- **Simplicidad:**
    Son fáciles de crear y mantener, ya que no requieren una base de datos o un backend complejo. 
- **Rendimiento:**
    Cargar una página estática es rápido, ya que no necesita procesamiento del lado del servidor para generar el contenido. 
- **Ejemplos:**
    Páginas de "Acerca de", portafolios, sitios de folletos, páginas de destino. 
    

Cómo crear una página web estática:
- **Codificar desde cero:**
    Utilizar HTML, CSS y JavaScript para crear manualmente los archivos de la página.
- **Generadores de sitios estáticos:**
    Utilizar herramientas que automatizan la creación de páginas estáticas a partir de plantillas. 

Ventajas:
- **Facilidad de desarrollo y mantenimiento:** Son más fáciles de construir y actualizar que los sitios dinámicos.
- **Rendimiento y velocidad:** Cargar una página estática es rápido, ya que no requiere procesamiento del lado del servidor.
- **Seguridad:** Los sitios estáticos son menos vulnerables a ataques informáticos, ya que no tienen un backend.
- **Asequibilidad:** Son más económicos de alojar que los sitios dinámicos. 

Desventajas:
- **Falta de interactividad:** No permiten la interacción del usuario para cambiar el contenido.
- **Dificultad para realizar cambios:** Actualizar el contenido requiere modificar los archivos HTML, lo que puede ser complicado para usuarios sin conocimientos de codificación.
- **Menos flexibilidad:** No son ideales para sitios web que requieren contenido dinámico o que cambian con frecuencia.
